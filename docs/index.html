A conductor for serious agent runs.진지한 에이전트 실행을 위한 지휘자.

Big single-agent runs drift. `/supergoal` keeps the work phased, reviewable, and gated by evidence.

큰 단일 에이전트 실행은 쉽게 흔들립니다. `/supergoal`은 작업을 단계화하고, 검토 가능하게 만들고, 증거로 게이트를 겁니다.
